# 前沿洞察

> 通过 blogwatcher/arxiv 收集的 LLM/Agent/MLOps 前沿洞察笔记

## 使用说明

- 每发现一篇值得记录的前沿文章/论文，写一条 insight
- 格式：日期 + 来源 + 核心观点 + 对我们的意义
- 每季度汇总一次

## 格式模板

```markdown
### YYYY-MM-DD — <文章标题>

**来源**: <URL>
**标签**: #agent #rag #multimodal #...
**重要度**: ⭐⭐⭐⭐

**核心观点**:
- 要点 1
- 要点 2

**对我们的意义**:
- 这个对我们广告平台的 Agent 开发有什么影响？
- 有什么可以借鉴的实践？
```
