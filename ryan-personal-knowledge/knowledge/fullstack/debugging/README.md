# 调试经验

> 系统化的调试方法论、工具使用、排障技巧

## 覆盖范围

- **系统性调试** — 4 阶段根因调试法：理解→假设→验证→修复
- **Python 调试** — pdb, debugpy (DAP), pytest-breakpoint
- **Node.js 调试** — --inspect + Chrome DevTools Protocol
- **浏览器排障** — console.log, DevTools Network/Elements 面板
- **网络排障** — curl, tcpdump, Charles Proxy

## 技能关联

- `systematic-debugging` — 4 阶段调试框架
- `python-debugpy` — Python 远程调试
- `node-inspect-debugger` — Node.js 调试
