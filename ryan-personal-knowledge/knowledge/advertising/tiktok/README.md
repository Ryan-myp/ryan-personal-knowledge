# TikTok Ads 知识

> TikTok 广告平台核心概念、API、数据流、最佳实践

## 覆盖范围

- **Spark Ads** — 有机视频素材广告化、创作者合作
- **Pangle** — TikTok 联盟网络、跨应用展示
- **For You Feed** — 推荐流广告、信息流原生
- **Marketing API** — 广告账户、广告系列管理、数据同步
- **Conversion API** — 服务器端数据回传
- **商品广告** — Shop 集成、商品 feed
