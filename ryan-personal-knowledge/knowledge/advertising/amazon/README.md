# Amazon Ads 知识

> Amazon 广告平台核心概念、API、最佳实践

## 覆盖范围

- **Sponsored Products (SP)** — 商品推广、搜索广告
- **Sponsored Brands (SB)** — 品牌推广、品牌广告位
- **Sponsored Display (SD)** — 展示广告、再营销
- **DSP (Demand-Side Platform)** — 程序化购买、展示/视频广告
- **Brand Analytics** — 品牌分析、搜索词报告
