# Microsoft Advertising 知识

## 覆盖范围

- **Bing Ads** — 搜索广告、展示广告
- **Microsoft Audience Network** — 跨微软生态展示
- **Shopping Ads** — 产品广告
- **Marketing API** — 广告账户管理、数据同步
