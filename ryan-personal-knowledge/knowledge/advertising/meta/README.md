# Meta Ads 知识

> Meta (Facebook/Instagram) 广告平台核心概念、API、数据流、最佳实践

## 覆盖范围

- **Advantage+** — 自动化广告系列、智能投放、素材优化
- **Campaign Structure** — 广告系列/广告组/广告层级结构
- **像素追踪 (Pixel)** — 转化追踪、自定义事件、服务器侧追踪 (CAPI)
- **Marketing API** — 广告账户管理、A/B 测试、数据同步
- **Audience Network** — 跨应用广告、受众拓展
