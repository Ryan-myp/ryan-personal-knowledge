# ad-ai-coding 知识

> 广告平台代码库知识沉淀，通过 ad-knowledge-query 工具自动发现和分析

## 覆盖范围

- **代码库架构** — 广告平台核心服务目录结构、调用链
- **API 文档** — 平台 API 定义、调用方式、参数说明
- **知识提取引擎** — ad-knowledge-query 工作原理、RRF 融合策略
- **意图路由** — 根据查询类型自动匹配 code/sqlite/knowledge_card
- **Known Pitfalls** — 已知查询坑和绕过方案

## 工具集成

- `ad-knowledge-query` — 查询当前实现逻辑
- `ad-knowledge-doc` — 生成知识库文档包
- `ad-test-case` — 自动生成测试用例
- `ad-test-automation` — 自动化测试计划
- `ad-dev` — 技术设计/开发计划转代码
