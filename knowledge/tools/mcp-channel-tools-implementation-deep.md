## 第五部分：自测题

### Q1: MCP 工具注册的三大要素？

**A**: 工具名称（唯一标识）、工具描述（功能说明）、输入参数（JSON Schema 定义）。

### Q2: 四大平台适配器的共同认证方式？

**A**: 都使用 OAuth2 认证、都有访问令牌刷新机制、都有权限范围控制。

### Q3: 二次确认机制的三个层面？

**A**: 操作分类（只读/普通/危险）、确认方式（文本/数字/签名）、确认流程（预览/提醒/执行）。
