## 一、执行流程

```
每天早上 9:00
    │
    ▼
推送今日学习任务（30min 阅读）
    │
    ▼
你学习（碎片时间）
    │
    ▼
你随时发消息："学完了" 或 "来考我"
    │
    ▼
我出题检验（深度问答）
    │
    ▼
你回答
    │
    ▼
我打分 + 给答案 + 评分 + 给出下次改进方向
    │
    ▼
记录到进度表（session_search）
```

## 二、每日学习计划（按成长路线图模块轮换）

```

第 13-14 周：Kafka（Broker/分区/消费者/Rebalance）

第 17-18 周：广告数据分析（A/B Test/归因/LTV/CAC）
第 19-20 周：大数据系统（Hadoop/Spark/Flink）
第 21-22 周：架构与工程（微服务治理/CAP/DistributedTransaction）
第 23-24 周：Agent大模型（LangChain/Multi-Agent/RAG架构）
第 25-26 周：DevOps（K8s/Prometheus/Grafana/OpenTelemetry）
```

## 六、使用方式

1. **手动触发**：每次你发消息说"今日学习"或"来考我"，我从当前模块取一题考你
2. **自动推送**：可以用 cron job 每天早上 9 点推送学习任务（需配置）
3. **灵活选择**：也可以指定模块，比如"考我 MySQL"
