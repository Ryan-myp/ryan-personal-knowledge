## 第二部分：亚马逊电商运营

### 亚马逊运营核心

```
亚马逊运营流程：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 选品策略                                                          │
│    ├── 市场分析：趋势和竞争分析                                     │
│    ├── 利润计算：成本/售价/佣金/物流                                │
│    └── 差异化：独特卖点                                             │
│                                                                     │
│ 2. Listing 优化                                                      │
│    ├── 标题：关键词+品牌+特性                                       │
│    ├── 图片：高清+多角度+场景图                                     │
│    ├── 描述：卖点+规格+使用说明                                     │
│    └── A+页面：品牌故事+对比图表                                    │
│                                                                     │
│ 3. 广告投放                                                          │
│    ├── Sponsored Products：商品推广                                 │
│    ├── Sponsored Brands：品牌推广                                   │
│    ├── Sponsored Display：展示推广                                  │
│    └── Video Ads：视频广告                                         │
│                                                                     │
│ 4. 库存管理                                                          │
│    ├── FBA：亚马逊物流                                              │
│    ├── 补货策略：基于销售预测                                       │
│    └── 库存健康：IPI 分数管理                                       │
└─────────────────────────────────────────────────────────────────────┘
```

亚马逊广告优化：

## 第三部分：英语语法基础

### 核心语法结构

```
英语语法基础：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 句子结构                                                          │
│    ├── 主谓宾：SVO 结构                                             │
│    ├── 主系表：SV+C 结构                                            │
│    └── 复合句：从句和连接词                                         │
│                                                                     │
│ 2. 时态系统                                                          │
│    ├── 现在时：一般/进行/完成/完成进行                             │
│    ├── 过去时：一般/进行/完成/完成进行                             │
│    ├── 将来时：一般/进行/完成/完成进行                             │
│    └── 过去将来时：would/do/would have done                        │
│                                                                     │
│ 3. 非谓语动词                                                        │
│    ├── 不定式：to do                                                │
│    ├── 动名词：doing                                                │
│    └── 分词：doing/done                                             │
│                                                                     │
│ 4. 虚拟语气                                                          │
│    ├── 条件句：if 引导的条件状语从句                                │
│    ├── 建议/命令：suggest/require that...                          │
│    └── 愿望：wish/if only...                                        │
└─────────────────────────────────────────────────────────────────────┘
```
