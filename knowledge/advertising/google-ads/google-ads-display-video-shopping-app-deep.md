# Google Ads 展示/视频/购物/应用广告深度实战

## 一、展示广告 (Display Ads) 深度解析

### 1.1 GDN 生态与投放原理

**Google 展示网络 (GDN) 规模：**
- 覆盖 200 万+ 网站和应用
- 触达全球 90%+ 互联网用户
- 每天展示超过 1 万亿次广告

**展示广告工作原理：**

```
展示广告竞价流程：
1. 用户访问 GDN 网站
2. 网站向广告请求广告
3. Google 分析用户画像
4. 匹配符合条件的广告
5. 实时竞价 (RTB)
6. 展示最优广告

用户画像数据源：
├── 浏览历史 (GDN 网站)
├── 搜索历史 (Google Search)
├── YouTube 观看记录
├── Gmail 内容 (历史数据)
├── 位置数据 (Google Maps)
├── 应用使用 (Google Play)
└── 第一方数据 (advertiser Pixel)
```

### 1.2 展示广告格式与规格

**广告格式对比：**

| 格式 | 尺寸 | 特点 | 适用场景 |
|------|------|------|----------|
| 矩形广告 | 300x250, 336x280 | 最常见，高填充率 | 品牌曝光 |
| 宽矩形广告 | 468x60, 728x90 | 顶部横幅，高可见度 | 品牌曝光 |
| skyscraper | 160x600, 120x600 | 侧边栏，长条形 | 品牌曝光 |
| 响应式广告 | 自适应 | 自动适配各种尺寸 | 高效投放 |
| HTML5 广告 | 自适应 | 富媒体交互 | 品牌互动 |

**响应式展示广告 (RDA) 工作原理：**

```
上传素材 → Google 自动生成 → 自动适配
├── 上传图片 → 各种尺寸组合 → 网站横幅
├── 上传 Logo → 不同颜色搭配 → App 广告
├── 添加标题 → 多种布局样式 → YouTube 侧边栏
└── 添加描述 → 自动优化 → 各种位置
```

### 1.3 展示广告定向策略

**定向方式对比：**

| 定向方式 | 说明 | 精准度 | 适用场景 |
|----------|------|--------|----------|
| 受众定向 | 基于兴趣和人口统计 | 中 | 品牌曝光 |
| 关键词定向 | 基于网页内容关键词 | 中高 | 相关网站投放 |
| 主题定向 | 基于网站主题 | 中 | 兴趣相关投放 |
| placements 定向 | 指定具体网站/App | 高 | 精准投放 |
| 再营销 | 针对网站访客 | 高 | 转化提升 |
| 类似受众 | 基于种子受众扩展 | 中 | 寻找新用户 |

**再营销受众分层策略：**

```
再营销受众分层：
├── 所有网站访客 (30 天)
│   ├── 展示品牌广告
│   └── 出价：基准
├── 产品页面访客 (30 天)
│   ├── 展示浏览过的产品
│   └── 出价：+20%
├── 购物车放弃用户 (7 天)
│   ├── 提供折扣优惠
│   └── 出价：+50%
├── 已购买用户 (365 天)
│   ├── 交叉销售相关产品
│   └── 出价：基准 (排除购买用户)
└── VIP 用户 (高价值)
    └── 独家优惠和新品预览
```

## 二、视频广告 (Video Ads) 深度解析

### 2.1 YouTube 广告生态

**YouTube 规模：**
- 月活用户超过 20 亿
- 每天观看超过 10 亿小时视频
- 覆盖 100+ 个国家，70+ 种语言

### 2.2 视频广告格式详解

**广告格式对比：**

| 格式 | 说明 | 时长 | 计费方式 | 适用场景 |
|------|------|------|----------|----------|
| Skippable In-Stream | 可跳过插播广告 | 5-300 秒 | CPV | 品牌故事 |
| Non-Skippable In-Stream | 不可跳过插播广告 | 15-30 秒 | vCPM | 品牌曝光 |
| Bumper Ads | 快闪广告 | ≤6 秒 | CPM | 频率控制 |
| Outstream | 外置视频广告 | 15-30 秒 | CPC/CPM | 移动优先 |
| Video Discovery | 视频发现广告 | - | CPC | 内容推广 |
| Shorts Ads | Shorts 广告 | ≤60 秒 | CPV | 年轻受众 |

**Skippable In-Stream Ads 工作流程：**

```
广告展示流程：
├── 视频播放前 (Pre-roll)
├── 视频中 (Mid-roll)
└── 视频后 (Post-roll)

用户交互：
├── 5 秒后可跳过
├── 完整观看 = 有效观看
├── 点击广告 = 点击
└── 关闭广告 = 无效果

优化目标：
├── 品牌认知 → 提高完整观看率
├── 网站流量 → 提高点击率
└── 转化 → 提高转化率
```

### 2.3 视频创意优化

**高效果视频结构：**

```
前 5 秒 (Hook)：
├── 抓住注意力
├── 展示品牌 Logo
└── 提出痛点或问题

中间部分 (Message)：
├── 传达核心价值
├── 展示产品特点
└── 建立情感连接

结尾部分 (CTA)：
├── 明确的行动号召
├── 优惠信息
└── 品牌记忆点
```

**视频广告优化指标：**

| 指标 | 说明 | 优化方向 |
|------|------|----------|
| 观看完成率 | 完整观看比例 | 优化前 5 秒、控制时长 |
| 点击率 (CTR) | 广告点击比例 | 优化 CTA、创意吸引力 |
| 每次观看费用 (CPV) | 平均观看成本 | 优化定向、竞价策略 |
| 品牌提升度 | 品牌认知提升 | 优化品牌露出、信息传达 |

## 三、购物广告 (Shopping Ads) 深度解析

### 3.1 购物广告工作原理

```
产品数据源 (Feed) → Google Merchant Center → Google Ads → 搜索结果展示
├── 产品 ID → 数据验证 → 广告系列创建 → 产品图片
├── 标题 → 政策审核 → 产品分组 → 标题
├── 描述 → 产品审核 → 出价设置 → 价格
├── 价格 → → → 商家信息
├── 图片 URL → → →
├── 可用状态 → → →
└── 目标国家 → → →
```

### 3.2 产品分组策略

**产品分组层级：**

```
All Products (所有产品)
├── Category: Electronics
│   ├── Brand: Apple
│   │   ├── Product Type: iPhone
│   │   └── Product Type: MacBook
│   └── Brand: Samsung
│       ├── Product Type: Galaxy
│       └── Product Type: Tablet
├── Category: Clothing
│   ├── Gender: Women
│   │   ├── Size: S
│   │   └── Size: M
│   └── Gender: Men
│       ├── Size: S
│       └── Size: M
└── Custom Labels
    ├── High Margin (label_0 = 1)
    ├── Sale Items (label_2 = 1)
    └── New Arrivals (custom_label_4 = 1)
```

### 3.3 产品 Feed 优化

**Feed 优化 checklist：**

```
标题优化：
├── [ ] 品牌 + 产品类型 + 关键属性
├── [ ] 包含搜索量高的关键词
└── [ ] 避免冗余信息

图片优化：
├── [ ] 白底高清图片
├── [ ] 多角度展示
└── [ ] 符合 Google 图片要求

属性优化：
├── [ ] 完整的 GTIN/MPN
├── [ ] 准确的价格和可用性
└── [ ] 详细的產品描述
```

## 四、应用广告 (App Ads) 深度解析

### 4.1 应用广告类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 应用安装广告 | 引导用户下载安装 | 新应用推广 |
| 应用内转化广告 | 引导用户完成特定操作 | 应用活跃 |
| 应用重新 engagement | 引导已安装用户重新打开 | 用户留存 |

### 4.2 应用广告优化

**优化指标：**

| 指标 | 说明 | 优化方向 |
|------|------|----------|
| CPI | 每次安装成本 | 优化定向、创意 |
| ROAS | 广告支出回报率 | 优化出价策略 |
| LTV | 用户终身价值 | 优化用户质量 |
| Retention | 用户留存率 | 优化用户体验 |
