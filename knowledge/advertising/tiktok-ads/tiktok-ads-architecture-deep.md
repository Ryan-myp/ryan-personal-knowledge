# TikTok Ads 平台架构与核心概念

## 一、TikTok 平台生态与市场格局

### 1.1 TikTok 是什么？

TikTok 是全球最受欢迎的短视频社交平台，月活用户超过 15 亿，覆盖 150+ 国家和地区。

**核心优势：**
- **年轻用户群体**：Z 世代和千禧一代为主，消费能力强
- **沉浸式短视频**：原生内容体验，广告干扰感低
- **强大算法推荐**：基于兴趣的精准推荐，用户停留时间长
- **创意形式多样**：短视频、直播、商城、搜索全覆盖

**市场规模：**
- 2023 年 TikTok 广告收入约 120 亿美元，同比增长 30%+
- 用户日均使用时长超过 95 分钟
- Gen Z 用户占比超过 60%

### 1.2 竞争格局对比

| 平台 | 月活用户 | 日均使用时长 | 核心优势 | 劣势 |
|------|----------|--------------|----------|------|
| TikTok | 15 亿+ | 95+ 分钟 | 年轻用户、短视频创新 | 数据追踪不完善 |
| YouTube | 25 亿+ | 40+ 分钟 | 长视频、搜索意图 | 广告跳过率高 |
| Instagram | 20 亿+ | 30+ 分钟 | 视觉社交、网红经济 | 有机触达下降 |
| Facebook | 30 亿+ | 20+ 分钟 | 用户基数大、定向精准 | 年轻用户流失 |

## 二、账户体系架构

### 2.1 账户层级结构

```
Business Center (商务中心)
├── Ad Accounts (广告账户)
│   ├── Campaigns (广告系列)
│   │   ├── Ad Groups (广告组)
│   │   │   ├── Ads (广告)
│   │   │   └── Creative Assets (创意素材)
│   │   └── Budget (预算)
│   └── Pixels (像素)
├── Users (用户)
├── Payment Methods (支付方式)
├── Identity (身份)
│   ├── TT User (TT 用户)
│   └── Auth Code (授权码)
└── Catalogs (商品目录)
```

### 2.2 Identity (身份) 详解

**两种类型：**

| 类型 | 说明 | 用途 |
|------|------|------|
| TT User | TikTok 用户身份 | 个人广告账户 |
| Auth Code | 授权码身份 | 代理商管理客户账户 |

**Auth Code 工作流程：**

```
1. 广告主登录 TikTok Ads Manager
2. 生成 Auth Code
3. 分享给代理商
4. 代理商使用 Auth Code 授权
5. 代理商可以管理广告账户
```

## 三、核心概念详解

### 3.1 广告系列 (Campaign)

**关键属性：**

| 属性 | 说明 | 示例 |
|------|------|------|
| Name | 广告系列名称 | "Summer_Sale_2024" |
| Objective | 优化目标 | Sales, App Promotion, Lead Generation |
| Budget | 每日预算 | $100/天 |
| Bidding Strategy | 出价策略 | Lowest Cost, Cost Cap, Bid Cap |
| Status | 状态 | ENABLED, PAUSED, UNDER_REVIEW |

**广告系列目标类型：**

| 目标 | 说明 | 适用场景 |
|------|------|----------|
| Sales (销售) | 促成网站或应用内购买 | 电商转化 |
| App Promotion (应用推广) | 推广移动应用 | 应用安装、应用内事件 |
| Lead Generation (潜在客户) | 收集潜在客户信息 | B2B、教育、金融 |
| Website Traffic (网站流量) | 引导用户访问网站 | 品牌曝光、引流 |
| Video Views (视频观看) | 提升视频观看量 | 品牌故事、内容推广 |
| Engagement (互动) | 提升帖子互动 | 品牌建设、社区运营 |

### 3.2 广告组 (Ad Group)

**关键属性：**

| 属性 | 说明 | 示例 |
|------|------|------|
| Name | 广告组名称 | "Women_18-35_Interests" |
| Placements | 投放位置 | Automatic, Manual |
| Budget | 每日预算 | $50/天 |
| Targeting | 受众定向 | 兴趣、行为、人口统计 |
| Bid Strategy | 出价策略 | Lowest Cost, Cost Cap |

**投放位置：**

| 位置 | 说明 | 适用广告类型 |
|------|------|-------------|
| TikTok Feed | TikTok 信息流 | In-Feed Ads |
| TikTok Search | TikTok 搜索 | Search Ads |
| TikTok Post | 视频发布后 | Spark Ads |
| TikTok Marketplace | TikTok 商城 | TikTok Shop Ads |
| TikTok Series | 系列视频 | 长视频广告 |
| TikTok Live | 直播间 | 直播购物广告 |

## 四、受众定向详解

### 4.1 定向方式

**基础定向：**

| 维度 | 选项 | 示例 |
|------|------|------|
| 年龄 | 13-65+ | 18-24, 25-34, 35-44 |
| 性别 | 所有、男性、女性 | 女性 18-35 |
| 语言 | 用户设备语言 | 中文、英语 |
| 国家/地区 | 投放国家 | 美国、中国、日本 |

**兴趣定向：**

| 类别 | 示例兴趣 |
|------|----------|
| 时尚美妆 | Beauty, Fashion, Skincare |
| 游戏娱乐 | Gaming, Entertainment, Music |
| 食品饮料 | Food, Cooking, Recipes |
| 运动健身 | Fitness, Health, Sports |
| 教育培训 | Education, Learning, Courses |

**行为定向：**

| 行为类型 | 说明 | 示例 |
|----------|------|------|
| 视频互动 | 点赞、评论、分享 | 高互动用户 |
| 达人互动 | 达人互动行为 | 达人粉丝 |
| 电商行为 | 浏览、购买 | 电商用户 |
| 应用行为 | 下载、使用 | 应用用户 |

### 4.2 自定义受众

**创建方式：**

| 方式 | 说明 | 数据要求 |
|------|------|----------|
| 客户列表 | 上传邮箱、手机号 | CSV 文件 |
| 网站活动 | Pixel 追踪 | 网站流量 |
| 应用活动 | SDK 追踪 | 应用用户 |
| 视频观看者 | 视频观看记录 | 观看数据 |
| 互动用户 | 帖子互动用户 | 互动数据 |

**再营销策略：**

```
再营销受众分层
├── 所有网站访客 (30 天)
│   └── 展示品牌广告
├── 产品页面访客 (30 天)
│   └── 展示浏览过的产品
├── 购物车放弃用户 (7 天)
│   └── 提供折扣优惠
├── 已购买用户 (365 天)
│   └── 交叉销售相关产品
└── VIP 用户 (高价值)
    └── 独家优惠和新品预览
```

### 4.3 类似受众

**创建原理：**

```
种子受众 (Seed Audience)
├── 客户列表
├── 网站访客
└── 应用用户
    ↓
TikTok 算法分析
├── 人口统计特征
├── 兴趣爱好
├── 行为模式
└── 社交关系
    ↓
类似受众 (Lookalike Audience)
├── 1% (最相似)
├── 2-5% (平衡)
└── 10% (最大范围)
```

## 五、竞价与计费

### 5.1 竞价策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| Lowest Cost (最低成本) | 在预算内获得最多结果 | 大多数场景 |
| Cost Cap (成本上限) | 控制平均每次结果成本 | 成本敏感 |
| Bid Cap (出价上限) | 设置最高出价 | 竞争激烈 |
| Target CPA (目标每次转化费用) | 设定目标每次转化费用 | 转化目标 |
| Target ROAS (目标广告支出回报率) | 设定目标广告支出回报率 | 收入最大化 |

### 5.2 计费方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| oCPM (目标千次展示) | 按千次展示付费，优化目标转化 | 大多数场景 |
| CPC (按点击付费) | 按点击付费 | 引流 |
| CPV (按观看付费) | 按视频观看付费 | 视频广告 |
| CPA (按转化付费) | 按转化付费 | 效果广告 |
