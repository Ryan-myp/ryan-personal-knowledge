# TikTok Ads 常见问题与排障深度实战

## 一、投放异常处理深度策略

### 1.1 展示量异常

**症状：** 展示量突然下降 50%+

**排查流程：**

```
Step 1: 检查预算
├── 每日预算是否耗尽？
│   ├── 是 → 提高预算或降低出价
│   └── 否 → 继续排查
└── 月度预算是否封顶？
    ├── 是 → 等待下月或提高月预算
    └── 否 → 继续排查

Step 2: 检查广告状态
├── 广告是否被拒绝？
│   ├── 是 → 查看拒绝原因，修改后重新提交
│   └── 否 → 继续排查
├── 广告系列是否暂停？
│   ├── 是 → 启用广告系列
│   └── 否 → 继续排查
└── 广告组是否暂停？
    ├── 是 → 启用广告组
    └── 否 → 继续排查

Step 3: 检查出价
├── 出价是否低于市场水平？
│   ├── 是 → 提高出价 10-20%
│   └── 否 → 继续排查
└── 质量评分是否下降？
    ├── 是 → 优化广告文案和落地页
    └── 否 → 继续排查

Step 4: 检查定向
├── 地理位置是否过于狭窄？
│   ├── 是 → 扩大地理范围
│   └── 否 → 继续排查
├── 受众是否过于小众？
│   ├── 是 → 扩大受众范围
│   └── 否 → 继续排查
└── 排期设置是否正确？
    ├── 是 → 保持
    └── 否 → 调整排期
```

**实际案例：**

```
案例：电商客户展示量下降 60%
├── 现象：
│   ├── 展示量从 8 万/天下降到 3.2 万/天
│   └── 花费从 $400/天下降到 $160/天
├── 排查：
│   ├── Step 1: 预算未耗尽 ✓
│   ├── Step 2: 广告状态正常 ✓
│   ├── Step 3: 出价未调整 ✓
│   └── Step 4: 发现创意疲劳
├── 根因：
│   ├── 同一视频运行超过 45 天
│   ├── CTR 从 2.5% 下降到 1.2%
│   └── 竞争加剧，同类广告增多
├── 解决方案：
│   ├── 更新 3 个视频创意
│   ├── 提高核心广告组出价 15%
│   └── 扩大兴趣定向范围
└── 结果：
    ├── 5 天后展示量恢复到 7 万/天
    ├── CTR 恢复到 2.3%
    └── 花费恢复到 $380/天
```

### 1.2 CTR 异常

**症状：** CTR 突然下降 30%+

**可能原因：**

| 原因 | 识别方法 | 解决方案 |
|------|----------|----------|
| 创意疲劳 | 同一广告运行 >30 天 | 更新创意 |
| 竞争加剧 | 竞品推出新广告 | 优化差异化 |
| 匹配变宽 | 广泛匹配引入不相关流量 | 添加否定关键词 |
| 位置变化 | 广告展示位置下降 | 提高出价或质量评分 |

**创意疲劳检测与应对：**

```
创意疲劳指标
├── CTR 下降 >20% (对比 30 天平均)
├── CPA 上升 >25%
└── 视频完播率下降 >30%

应对策略
├── 更新视频开头 3 秒
├── 更换背景音乐
├── 更新文案和 CTA
├── 测试新达人合作
└── 暂停低效创意
```

### 1.3 CPA 异常

**症状：** CPA 突然上升 50%+

**排查流程：**

```
Step 1: 检查转化追踪
├── Pixel 是否正常？
│   ├── 测试转化事件
│   └── 检查数据回传
├── 转化窗口是否变化？
│   └── 确认设置
└── 是否有新增转化来源？
    └── 检查报告

Step 2: 检查流量质量
├── 搜索词报告是否有低质流量？
│   └── 添加否定关键词
├── 地理位置是否有变化？
│   └── 调整地域出价
└── 设备分布是否变化？
    └── 调整设备出价

Step 3: 检查竞争环境
├── 竞品是否提高出价？
│   └── 分析竞品广告
├── 季节性因素？
│   └── 调整预算和出价
└── 市场变化？
    └── 优化价值主张
```

## 二、审核拒绝处理

### 2.1 常见拒绝原因

**广告内容拒绝：**

| 原因 | 说明 | 解决方法 |
|------|------|----------|
| 内容违规 | 含敏感内容 | 修改视频、替换内容 |
| 音乐版权 | 使用未授权音乐 | 使用 TikTok 音乐库 |
| 品牌侵权 | 未经授权的品牌使用 | 获得授权、移除品牌 |
| 产品质量 | 产品不符合政策 | 检查产品合规性 |
| 落地页问题 | 页面无法访问 | 修复页面、更新 URL |

**政策合规拒绝：**

| 类别 | 限制内容 | 替代方案 |
|------|----------|----------|
| 金融产品 | 承诺收益 | 展示产品功能 |
| 健康产品 | 疗效承诺 | 展示成分 |
| 美容产品 | 前后对比 | 展示使用效果 |
| 教育产品 | 保证结果 | 展示课程内容 |

### 2.2 审核申诉流程

```
1. 查看拒绝原因
   ↓
2. 分析问题根源
   ├── 视频内容问题
   ├── 音乐版权问题
   └── 政策合规问题
   ↓
3. 修改广告内容
   ↓
4. 重新提交审核
   ↓
5. 等待审核结果 (通常 1-3 天)
   ↓
6. 如仍被拒，使用申诉通道
```

**申诉技巧：**

```
申诉材料准备
├── 修改说明
│   ├── 明确指出修改内容
│   ├── 说明为何符合政策
│   └── 提供证据材料
├── 政策引用
│   ├── 引用相关政策条款
│   ├── 说明合规性
│   └── 提供法律依据
└── 联系方式
    ├── 提供联系人
    ├── 提供电话/邮箱
    └── 承诺及时响应
```

## 三、数据追踪问题

### 3.1 Pixel 配置优化

**Pixel 事件追踪：**

```
标准事件
├── PageView
├── ViewContent
├── AddToCart
├── InitiateCheckout
└── Purchase

自定义事件
├── 视频观看
├── 达人互动
└── Spark Ads 点击

事件优化
├── 事件匹配优化
├── 转换值设置
└── 归因窗口期调整
```

**事件匹配优化：**

| 方法 | 说明 | 效果 |
|------|------|------|
| 哈希匹配 | 发送哈希处理的邮箱/手机 | 提高匹配率 20-30% |
| 设备指纹 | 收集设备信息 | 提高识别率 |
| 用户代理 | 浏览器/User-Agent | 辅助匹配 |
| IP 地址 | 网络地址 | 辅助匹配 |

### 3.2 iOS 隐私影响应对

**iOS 14+ 影响：**

```
影响分析
├── 转化追踪延迟 (最多 3 天)
├── 转化丢失 (30-50%)
└── 受众规模缩小

应对措施
├── 使用 Conversion API
├── 验证域名 (Domain Verification)
├── 优先使用事件匹配 (Event Match)
├── 使用 Aggregated Event Measurement
└── 调整归因窗口期
```

**Conversion API 配置：**

```
配置步骤
1. 设置服务器端追踪
   ├── 安装 SDK
   ├── 配置事件
   └── 测试数据回传
2. 优化事件匹配
   ├── 发送用户数据 (邮箱、手机)
   ├── 哈希处理
   └── 验证匹配率
3. 监控数据一致性
   ├── 对比 Pixel 和 CA 数据
   ├── 分析差异原因
   └── 优化匹配策略
```

## 四、常见问题快速诊断

### 4.1 问题诊断决策树

```
问题诊断流程
├── 展示量低？
│   ├── 预算耗尽 → 提高预算
│   ├── 出价过低 → 提高出价
│   ├── 定向过窄 → 扩大定向
│   └── 审核拒绝 → 修改内容
├── CTR 低？
│   ├── 创意疲劳 → 更新创意
│   ├── 文案不佳 → 优化文案
│   ├── 定向不准 → 调整定向
│   └── 竞争激烈 → 差异化
├── CPA 高？
│   ├── 落地页差 → 优化落地页
│   ├── 出价过高 → 降低出价
│   ├── 流量质量差 → 优化定向
│   └── 转化追踪问题 → 检查 Pixel
└── 转化量低？
    ├── 流量不足 → 扩大定向
    ├── 转化率低 → 优化落地页
    ├── 出价过低 → 提高出价
    └── 追踪问题 → 检查 Pixel
```

### 4.2 性能基准参考

**各指标健康范围：**

| 指标 | 优秀 | 良好 | 需优化 |
|------|------|------|--------|
| CTR | >2% | 1-2% | <1% |
| CVR | >5% | 2-5% | <2% |
| CPA | <目标值 | 目标值 ±20% | >目标值 20% |
| ROAS | >400% | 250-400% | <250% |
| 完播率 | >50% | 30-50% | <30% |
