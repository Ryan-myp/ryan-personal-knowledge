# AI 安全与对抗样本深度：从模型鲁棒性到隐私保护

> 从对抗攻击原理到防御策略，逐层解析 AI 系统安全核心问题

---

## 第一部分：AI 安全威胁

### 对抗攻击原理

```
对抗攻击分类：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 白盒攻击                                                          │
│    ├── 梯度信息完全可知                                               │
│    ├── FGSM：快速梯度符号法                                         │
│    └── PGD：投影梯度下降                                            │
│                                                                     │
│ 2. 黑盒攻击                                                          │
│    ├── 仅能查询模型输出                                               │
│    ├── 迁移攻击：利用模型相似性                                     │
│    └── 查询攻击：逐步逼近决策边界                                   │
│                                                                     │
│ 3. 物理世界攻击                                                      │
│    ├── 对抗补丁：贴在物体上的贴纸                                   │
│    ├── 对抗眼镜：改变人脸识别结果                                   │
│    └── 对抗海报：误导自动驾驶系统                                   │
└─────────────────────────────────────────────────────────────────────┘
```

### 常见攻击手法

```
典型攻击场景：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 模型窃取                                                          │
│    ├── 通过大量查询重建模型                                         │
│    ├── 蒸馏攻击：训练替代模型                                       │
│    └── 模型反演：从输出推断训练数据                                 │
│                                                                     │
│ 2. 数据投毒                                                          │
│    ├── 训练阶段注入恶意样本                                         │
│    ├── 后门攻击：特定触发器导致错误输出                             │
│    └── 标签翻转：修改训练标签                                       │
│                                                                     │
│ 3. 隐私泄露                                                          │
│    ├── 成员推断：判断数据是否在训练集中                             │
│    ├── 属性推断：推断敏感属性                                       │
│    └── 差分隐私：添加噪声保护隐私                                   │
└─────────────────────────────────────────────────────────────────────┘
```

## 第二部分：对抗防御

### 防御策略

```
对抗防御方法：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 对抗训练                                                          │
│    ├── 训练时加入对抗样本                                           │
│    ├── 提高模型鲁棒性                                               │
│    └── 计算开销大                                                   │
│                                                                     │
│ 2. 输入预处理                                                        │
│    ├── 去噪：去除对抗扰动                                           │
│    ├── 压缩：减少高频信息                                           │
│    └── 随机化：增加攻击难度                                         │
│                                                                     │
│ 3. 检测机制                                                          │
│    ├── 异常检测：识别可疑输入                                       │
│    ├── 置信度监控：低置信度预警                                     │
│    └── 一致性检查：多次推理结果对比                                 │
│                                                                     │
│ 4. 形式化验证                                                        │
│    ├── 边界证明：数学证明模型鲁棒性                                 │
│    ├── 区间界限分析                                                 │
│    └── 计算复杂度高                                                 │
└─────────────────────────────────────────────────────────────────────┘
```

### 隐私保护技术

```
隐私保护方案：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 差分隐私                                                          │
│    ├── 训练时添加噪声                                                 │
│    ├── ε-差分隐私：隐私预算控制                                     │
│    └── 适用：统计查询、联邦学习                                     │
│                                                                     │
│ 2. 联邦学习                                                          │
│    ├── 数据不出本地                                                   │
│    ├── 模型参数聚合                                                   │
│    └── 适用：跨机构协作                                             │
│                                                                     │
│ 3. 同态加密                                                          │
│    ├── 密文计算                                                       │
│    ├── 全同态：任意计算                                               │
│    └── 部分同态：乘法或加法                                         │
│                                                                     │
│ 4. 安全多方计算                                                      │
│    ├── 多方联合计算                                                   │
│    ├── 不泄露各自输入                                                 │
│    └── 适用：联合统计                                               │
└─────────────────────────────────────────────────────────────────────┘
```

## 第四部分：自测题

### Q1: 对抗攻击的三大分类？

**A**: 白盒攻击（梯度已知）、黑盒攻击（仅查询）、物理世界攻击（贴纸/眼镜）。

### Q2: 差分隐私的核心概念？

**A**: ε-隐私预算、训练时添加噪声、控制隐私泄露程度。

### Q3: AI 安全的四大防御策略？

**A**: 对抗训练、输入预处理、检测机制、形式化验证。
