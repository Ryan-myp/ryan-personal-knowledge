## 第一部分：AI 安全威胁

### 对抗攻击原理

```
对抗攻击分类：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 白盒攻击                                                          │
│    ├── 已知模型架构和参数                                           │
│    ├── FGSM（快速梯度符号法）                                       │
│    └── PGD（投影梯度下降）                                          │
│                                                                     │
│ 2. 黑盒攻击                                                          │
│    ├── 仅知道输入输出                                               │
│    ├── 查询攻击：通过查询推断模型                                   │
│    └── 迁移攻击：利用替代模型生成对抗样本                           │
│                                                                     │
│ 3. 物理世界攻击                                                      │
│    ├── 对抗补丁：贴在物体上的特殊图案                               │
│    ├── 对抗眼镜：改变人脸识别结果                                   │
│    └── 对抗交通标志：导致自动驾驶误判                               │
└─────────────────────────────────────────────────────────────────────┘
```

## 第三部分：防御策略

```
对抗训练：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 原理                                                              │
│    ├── 在训练过程中加入对抗样本                                     │

│    ├── 鲁棒性提升：对抗攻击成功率下降                               │

```

### 输入净化

```
输入净化：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 原理                                                              │
│    ├── 检测并移除输入中的对抗噪声                                   │

│    └── 提高模型鲁棒性                                               │
│                                                                     │

│    └── 随机化：对输入进行随机扰动                                   │
│                                                                     │

└─────────────────────────────────────────────────────────────────────┘
```

## 第四部分：自测题

### Q1: 对抗攻击的三大分类？

**A**: 白盒攻击（已知模型参数）、黑盒攻击（仅知道输入输出）、物理世界攻击（现实世界对抗样本）。

### Q2: 模型窃取攻击的三种防御策略？

**A**: 查询限制（限制 API 调用频率）、差分隐私（添加噪声保护隐私）、模型水印（嵌入版权标识）。

### Q3: 对抗训练的三大方法？

**A**: PGD 对抗训练（使用 PGD 生成对抗样本）、TRADES（平衡准确性和鲁棒性）、FreeLB（高效对抗训练）。
