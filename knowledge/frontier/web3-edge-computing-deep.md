## 第一部分：区块链基础

### 区块链架构

```
区块链核心组件：
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 区块                                                              │
│    ├── 区块头：版本号、时间戳、Merkle Root                           │
│    ├── 交易列表：该区块包含的交易                                   │
│    └── 前驱哈希：指向上一个区块                                       │
│                                                                     │
│ 2. 共识机制                                                          │
│    ├── PoW：工作量证明                                               │
│    ├── PoS：权益证明                                                 │
│    ├── DPoS：委托权益证明                                           │
│    └── PBFT：实用拜占庭容错                                         │
│                                                                     │
│ 3. 智能合约                                                          │
│    ├── 以太坊 EVM：以太坊虚拟机                                     │
│    ├── Solidity：合约编程语言                                       │
│    └── WebAssembly：通用执行环境                                   │
└─────────────────────────────────────────────────────────────────────┘
```


## 第四部分：自测题

### Q1: 区块链的三大核心组件？

**A**: 区块（区块头+交易+前驱哈希）、共识机制（PoW/PoS/DPoS/PBFT）、智能合约（EVM/Solidity/WASM）。

### Q2: 边缘计算的三层架构？

**A**: 云端（全局处理/模型训练）、边缘层（本地推理/数据过滤）、设备层（数据采集/实时控制）。

### Q3: Web3 + 边缘计算的应用场景？

**A**: 资源市场（闲置算力出租）、数据隐私（零知识证明）、dApp 架构（IPFS + Oracles）。
