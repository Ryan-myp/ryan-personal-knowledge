### 窗口函数实战

```sql
-- 广告数据窗口函数示例
SELECT 
    campaign_id,
    date,
    spend,
    impressions,
    clicks,
    -- 移动平均
    AVG(impressions) OVER (
        PARTITION BY campaign_id 
        ORDER BY date 
        ROWS BETWEEN 6 PRECEDING AND CURRENT ROW
    ) AS moving_avg_impressions,
    -- 排名
    RANK() OVER (
        PARTITION BY DATE_TRUNC('month', date)
        ORDER BY spend DESC
    ) AS spend_rank_in_month,
    -- 累计值
    SUM(spend) OVER (
        PARTITION BY campaign_id 
        ORDER BY date 
        ROWS UNBOUNDED PRECEDING
    ) AS cumulative_spend
FROM ad_performance
WHERE date >= '2024-01-01';
```
