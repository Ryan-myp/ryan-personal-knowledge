## 🔧 实践项目

### 1. 基础 Agent 实现
实现一个基于 LLM 的简单 Agent，支持：
- 意图识别
- 工具调用
- 任务执行

### 2. RAG 系统集成
实现一个 RAG 系统，包括：
- 文档索引
- 语义搜索
- 答案生成

### 3. Agent 设计模式
应用常见的 Agent 设计模式：
- 单 Agent 模式
- 多 Agent 协作模式
- 层次化 Agent 模式
